# Python Training Materials for Beginners

This folder contains 20 Python training files designed to teach newcomers various essential Python concepts including web development, databases, and other important topics.

## File List:

1. `01_basic_syntax.py` - Basic Python syntax and variables
2. `02_data_types.py` - Python data types and structures
3. `03_control_flow.py` - Conditions and loops
4. `04_functions.py` - Creating and using functions
5. `05_lists_tuples.py` - Working with lists and tuples
6. `06_dictionaries_sets.py` - Dictionaries and sets
7. `07_file_handling.py` - File operations
8. `08_error_handling.py` - Exception handling
9. `09_classes_objects.py` - Object-oriented programming basics
10. `10_modules_packages.py` - Importing and creating modules
11. `11_string_manipulation.py` - String operations and methods
12. `12_datetime_operations.py` - Working with dates and times
13. `13_list_comprehensions.py` - List comprehensions and generators
14. `14_lambda_functions.py` - Lambda functions and functional programming
15. `15_web_requests.py` - Making HTTP requests with requests library
16. `16_json_handling.py` - Working with JSON data
17. `17_sqlite_database.py` - SQLite database operations
18. `18_flask_basics.py` - Flask web framework basics
19. `19_api_development.py` - Building simple REST APIs
20. `20_project_integration.py` - Integrating concepts in a complete project
